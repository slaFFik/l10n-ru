Allows to get Russian translations for your plugins automatically.

== Description ==

Allows to get Russian translations for your plugins automatically from the [http://code.google.com/p/l10n-ru/ l10n-ru] project repository.

== Installation ==

1. Upload `rurumo` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
